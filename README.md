# react-native-escpos-android

A library to use escpos on android

## Installation

```sh
npm install react-native-escpos-android
```

## Usage

```js
import EscposAndroid from "react-native-escpos-android";

// ...

const result = await EscposAndroid.multiply(3, 7);
```
